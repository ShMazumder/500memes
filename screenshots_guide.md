## 2. How to Take Screenshots

Since the app is built with Flutter, you can easily take high-quality screenshots.

### Option A: Using Flutter Command (Best Quality)
1.  **Run the app** on your emulator or connected device:
    ```bash
    flutter run
    ```
2.  **Take a screenshot** by creating a new terminal tab (keep the app running) and typing:
    ```bash
    flutter screenshot
    ```
    This saves a screenshot file (e.g., `flutter_01.png`) in your project root.

### Option B: Using Android Emulator
1.  Run the app in the Android Emulator.
2.  Click the **Camera icon** ðŸ“· in the emulator toolbar (usually on the right).
3.  The image is saved to your desktop or pictures folder.

### Option C: On a Real Device
1.  Run the app on your phone.
2.  Press **Power + Volume Down** (standard Android combo).
